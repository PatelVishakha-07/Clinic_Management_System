# Clinic Management System

A desktop application built to digitize day-to-day operations for a clinic — patient records, prescriptions, billing, and profit tracking — replacing manual, paper-based record-keeping.

**Status: Deployed and in active use by a practicing doctor.**

## Overview

Clinics often rely on paper registers or basic spreadsheets to manage patients, medicines, and bills, which makes record retrieval slow and profit tracking unreliable. This system was built to solve that problem with a simple, reliable desktop application backed by a proper relational database.

## Features

- **Patient Record Management** — add, update, and search patient records quickly
- **Medicine Inventory Management** — track medicine stock and details
- **Prescription & Medical History** — store prescriptions and past medical history per patient
- **Billing & Payments** — generate bills and maintain payment records
- **Profit Tracking** — store and review profit history over time

## Tech Stack

| Layer | Technology |
|---|---|
| Application | C# (.NET, Windows Forms) |
| Database | PostgreSQL |
| IDE | Visual Studio 2022 |

## Getting Started

### Prerequisites
- Visual Studio 2022 (or later) with .NET desktop development workload
- PostgreSQL installed locally or accessible remotely

### Setup
1. Clone the repository
   ```bash
   git clone https://github.com/PatelVishakha-07/Clinic_Management_System.git
   ```
2. Open the solution file (`.sln`) in Visual Studio 2022
3. Update the PostgreSQL connection string in the configuration file with your local database credentials
4. Run the database setup scripts (if included) to create the required tables
5. Build and run the project from Visual Studio
